Portfolio for COP 4813: Internet Programming


Assignment 1

Assignment 1 was designed to teach us the fundamentals of HTML and CSS so that we would have a firm foundation to base to build upon when we delve deeper into the class of internet programming.

The HTML portion of the assignment required us to create a static page utilizing headers, email to links, images, hyper links, and tables. I also added divs to better seperate each section and to give the website more clear cut idea of where each section ended.

The CSS portion of the assignment required us to modify the appearance of multiple items and sections of the page to give it a unique but cohesive look that would set it apart from other pages online.


Learning Objectives

  • Create basic HTML pages utilizing the design practices learned from the material
  • Create a basic style sheet to apply a cohesive and unique feel to the HTML pages creted.
  • Learning the folder structure and how to navigate a Linux server through secure shell

Assignment 2

Assignment 2 was designed to teach us the fundamentals of Javascript by creating a form utilizing text boxes, radio buttons, drop down menues, and functions that process all the information provided through those options.


Learning Objectives

  • Create basic forms to collect information
  • Use textboxes, radio buttons, check boxes, and drop down menues
  • Utilize a reset button and submit buttion to modify the information in the form
  • Create a flow chart that illustrate the decision structure of the script
Assignment 3

Assignment 3 was designed to teach us the more advanced javascript techniques of manipulating arrays, creating functions, utilzing user input, and providing results from the user input using the functions created.


Learning Objectives

  • Create basic forms to collect information from user
  • Create mathematical functions that used the user input
  • Provide the results of their input in designated textboxes
Assignment 4

Assignment 4 was about introducing the canvas element and how to create and manipulate animations with what we've learned thus far. I chose to create a paint canvas to emphasize the creativity within the field of development and how design, aesthetic design, is just as important as technical knowledge.


Learning Objectives

  • Create animation with the html5 canvas tag
  • Utilize javascript, or jquery, to execute the canvas functions.
Assignment 5

Assignment 5 introduced PHP and server side programing. We had to create a basic log in page that verified a user's credentials and allowed them into a stock portfolio. From there they will be able to add/modify/delete stocks from their portfolio. I used a drop down menu with a select number of stocks so that the user wouldn't have to go look up company stock symbols to test the functionality of the code.
